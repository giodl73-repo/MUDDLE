# MUDDLE Research

Use this skill before adopting broad MUD/MUSH/IF, roguelike, terminal UI, or
adapter patterns.

## Gate criteria

- Cite local MUDDLE docs and target host repos.
- Keep RALLY/MUDDLE responsibilities separate.
- Avoid adopting networking, terminal UI frameworks, or persistence formats
  before a host repo proves the need.
- Record recommendations as adopt now, prototype behind boundary, or defer.
